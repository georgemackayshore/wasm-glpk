# WASM-GLPK

Bindings for compiling GLPK-5.0 into WebAssembly, using emscripten and cmake.
Inspired by node-glpk (https://github.com/hgourvest/node-glpk)

## Prerequisites

* emscripten SDK
* cmake v3.13+

## Build steps

1. mkdir build
2. cd build
3. emcmake cmake ..
4. make
